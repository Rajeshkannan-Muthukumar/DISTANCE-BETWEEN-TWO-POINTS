### PROGRAM:
```
  #Program to find the distance between two points:
#Developed by: Rajeshkannan.M

import math
x=[10,6]
y=[4,2]
dist=math.sqrt(((x[0]-x[1])**2)+((y[0]+y[1])**2))
print("{:.2f}".format(dist))
```
